HTML Structure

The template is based on Bootstrap 3 the most popular framework for developing responsive, mobile first projects.

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options, as well as powerful mixins for generating more semantic layouts.

Create a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding, then create .row and add columns with appropriate device name (xs, sm, md, lg) and grid number (2, 4, 6, 12).

For full documentation, please visit Bootstrap 3 site: http://getbootstrap.com/css/ 

<div class="container">
  <div class="row">
    <div class="col-md-12">
      ...
    </div>
  </div>
</div>

CSS Structure

Well organized and labeled CSS files is one of my priorities.
These are the CSS file you're be using in the template:

  • bootstrap.css (Generic styles and the basic layout structure styles)
  • style.css (Elements such as titles, sections, backgrounds, column styles, etc...)
  • css/colors/.css (template color skins)

  • blue.css

To use one of pre-made color skin that you like,
change the name of current linked color css file in the head section of HTML file: 

<link rel="stylesheet" href="css/colors/cyan.css" id="color-skins"/>

Also you make a new one by duplicating any of color css file and change the colors hex code.

Backgrounds

Some of the sections have backgrounds with parallax effect and to change those backgrounds,
simply open style.css and on section id change the link of background:


style.css: 

section.client-bg-img {
   background: url(../images/client-bg-img.png) repeat 50% 50%!important

section.onepage-blog {

Portfolio

A powerful jQuery plugin that provides beautiful animated filtering, custom captions and it's perfect for portfolios, galleries, team members, blog posts or any other ordered content with dynamic and responsive layouts.

Creating portfolio is quite easy following just two simple steps.
1) Add Portfolio Item to Section
2) Create Portfolio Item Description Page

Adding Portfolio Items to the Section

Find the below code and add it to portfolio list

HTML Markup for project item: 

<div class="portfolio-hover-div">
	<figure>
		<img class="img-responsive" src="images/portfolio/portfolio-8.jpg" alt="Image">
		<figcaption>
			<div class="details-link-info">
				<ul>
					<li><a href="#"><i class="fa fa-heart"></i>135 Likes</a></li>
					<li><a href="images/portfolio/portfolio-8.jpg" data-lightbox="portfolio-8.jpg"><i class="fa fa-expand"></i>View Larger</a></li>
				</ul>
			</div>
		</figcaption>
	</figure>
	<div class="portfolio-bottom-title">
		<h4>Project Title</h4>
		<p>Web Design, Product Design</p>
	</div>
</div>

Contact Form (PHP Script)

Only thing you need to do to make working contact form is to setup your email address.
To do this, please open "php/contact.php" file and change the email address on line 46:


contact.php: 

46   $address = "your-email-address@your-domain.com";
One notice: this will be functional only on your server, php file can not be executed on your local computer.
Upload to your server when testing.
